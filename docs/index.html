âœï¸ Getting Started

Generate random keys

npx noskey

Generate a vanity key

npx noskey --vanity 123

Generate a from private key

npx noskey -p 123

Install locally

sudo npm install -g noskey

Usage

Usage: noskey [options]

Options:
      --version  Show version number
  -v, --vanity   Vanity string
  -p, --priv     Private key
  -s, --nsec     From nsec
  -h, --help     Show help
